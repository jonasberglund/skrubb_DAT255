# User Stories
- As a user, I want to see "dagens lunch"
- As a user, I want to be able to see when the pub is open
- As a user, I want to be able to send fault reports in the app
- As a user, I want to see news when I open the app
- Documentation

# Hur det gick
Till en början verkade det som att "dagens lunch" kunde bli lite klurig, då vi inte fick till
något bra sätt att läsa av lunchdata. Det löste sig till slut med hjälp av Javas stränghantering.
Dock märkte vi lite i efterhand att sidan där vi hämtar lunchdata inte uppdateras nog frekvent
för att kunna användas dagligen.  Detta kommer förmodligen att ändras i nästa sprint.

Pubvyn tog inte allt för lång tid att få till eftersom det fanns en liknande sen innan (events).

Felrapporteringen var också ganska enkel att få till, lite som "Förslagslådan", men med lite mer info.

News är mer eller mindre klar, lite puts.

Det har också skrivits lite dokumentation och kommentarer. Dokumentationen är dels för utvecklare
men också för "kunden" (H-sektionens styrelse).
